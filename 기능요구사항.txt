기능요구사항

1. 상품관리/상품조회(송혜원)
-판매자는 상품을 등록, 수정, 조회, 삭제 가능
-관리자는 상품을 삭제 가능
-상품 목록 조회 가능
-상품 하나를 선택하면 상세 페이지로 이동
-상세페이지에서 장바구니담기, 바로구매 선택 가능

2. 장바구니(cart) + 결제(이대현)
-상세페이지에서 장바구니 담기를 선택하면 장바구니로 이동, 계속 쇼핑하기 선택 가능
-장바구니로 이동, 바로구매를 선택하면 결제페이지로 이동(로그인을 안 했을 경우 로그인 창으로 이동)
-구매한 상품은 sell Table에 정보 등록
-구매한 개수 만큼  Prodcut Table에서 해당 상품의 수량 감소

3.게시판(박현명)
공지게시판/문의게시판
-게시글 등록 
-게시글 수정 : 작성자 수정 가능
-게시글 삭제 : 작성자, 관리자 삭제 가능
-게시글 조회 : 조회하면 게시글 목록에서 조회수가 올라감.

4.회원관리(이나경)
-회원 본인이 마이페이지 수정, 탈퇴
-회원 본인이 내 결제 내역 확인
-관리자가 회원 수정, 삭제
-관리자가 판매자 관리
